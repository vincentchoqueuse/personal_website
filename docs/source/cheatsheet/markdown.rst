Markdown
========


Text Style
----------

Heading	
+++++++

.. code ::

    # H1
    ## H2
    ### H3


Bold
++++

.. code ::

    **bold text**

Italic
++++++

.. code ::

    *italicized text*

Blockquote
++++++++++

.. code ::

    > blockquote

List
----

Ordered List
++++++++++++

.. code ::

    1. First item
    2. Second item
    3. Third item

Unordered List
++++++++++++++

.. code ::

    - First item
    - Second item
    - Third item

Others 
------

Code
++++

.. code ::

    `code`

Horizontal Rule
+++++++++++++++

.. code ::

    ---

Link 
++++

.. code ::

    [title](https://www.example.com)


Image
+++++

.. code ::

    ![alt text](image.jpg)
